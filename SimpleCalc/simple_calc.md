# Simple Calc

A simple program performing basic operations on two given numbers.

## To compile

````shell
gcc -o SimpleCalc SimpleCalc.c
````

## To run

````shell
$ ./SimpleCalc 
Welcome to Simple Calc v1!
Type a number: 8
Type a second number: 2
8 + 2 = 10
8 - 2 = 6
8 * 2 = 16
8 / 2 = 4
8 mod 2 = 0
````
